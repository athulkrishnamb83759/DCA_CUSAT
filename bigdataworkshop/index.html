Course Coverage

The programme will cover the following topics:

  • Tools and techniques for big data analytics
  • Machine learning for big data analytics
  • NoSQL databases
  • Handling big data streams
  • Visualization techniques for big data analytics
  • Bigdata and security
  • Blockchain and big data
  • Cloud based big data frameworks
  • Hands-on using pyspark, Kafka, Cassandra, Hive
  • Applications of big data
